CHATBOT BITU - LOCAL (Flask)

Instrucciones rápidas:

1) Requisitos:
   - Python 3.8+
   - pip

2) Instalar dependencias:
   pip install flask

3) Ejecutar:
   python app.py

4) Abrir en el navegador:
   http://localhost:5000

El chat (Bitu) aparece como un ícono redondo abajo a la derecha. El contenido de preguntas frecuentes está en data/faqs.json.
Cuando Bitu no encuentra respuesta, muestra un mensaje con contacto: +54 9 11 5185-7753

Podés personalizar estilos en static/styles.css y las FAQs en data/faqs.json
